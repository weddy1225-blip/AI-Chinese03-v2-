# 🎨 文字調色盤 (Word Palette) - SEL 擴充版

這是一個專為 **國小三、四年級學生** 設計的互動式 AI 語文與情緒覺察工具。透過具備強烈畫面感的 67 組優美金句，引導學生練習將「文字意境」轉化為「視覺色彩」，並結合 **SEL (社會情緒學習)** 標籤，讓孩子在調色的同時進行自我覺察。

### 🌟 遊戲設計核心

#### 1. 豐富的意境題庫 (共 67 題)
系統內建 67 組精心挑選的句子，範疇涵蓋：
*   **微觀觀察**：例如「清晨的露珠躺在嫩葉中央」。
*   **擬人想像**：會跳舞的夕陽、穿著西裝的燕子、會嘆氣的小鐵門。
*   **生活美學**：圖書館的書香、紅磚牆的藤蔓、老時鐘的滴答聲。
*   **純粹意境**：句子中**嚴格限制不出現顏色名稱**，完全交由學生透過感官描寫來想像色彩。

#### 2. 直覺式 SEL 標籤互動 (無輸入框設計)
為了讓學生專注於感受而非打字，我們將原本的輸入框改為**「互動式標籤」**：
*   **三大情緒維度**：分為【活力與喜悅】、【平靜與神祕】、【情感與覺察】三大區塊，提供超過 18 種細膩的情緒標籤。
*   **操作邏輯**：點擊標籤即可選取並變色（黃色高亮），再次點擊即可取消，操作極度直覺。
*   **自我覺察**：透過標籤點選，引導學生思考文字帶來的心理價值（如：這句話帶給我的是「生機」還是「治癒」？）。

#### 3. 溫柔的 AI 教師點評
後端串接 **Gemini 1.5 Flash** 模型，扮演色彩心理學與國文老師：
*   **精確分析**：具體分析學生選擇的「色彩碼」與「心情標籤」如何與文字產生連結。
*   **暖心鼓勵**：針對學生的情緒覺察力給予肯定，並以適合中年級學生的自然語氣進行點評。
*   **去格式化**：評論內容親切自然，不含任何冒號或多餘符號，提升閱讀親和力。

### 🛠 技術架構
*   **前端介面**：HTML5 / CSS3（採用適合低年級的粉色調與大字體設計，支援響應式網頁）。
*   **後端邏輯**：Google Apps Script (GAS)。
*   **AI 引擎**：Google Gemini 1.5 Flash。
*   **資料安全**：API Key 儲存於 GAS 指令碼屬性 (Script Properties) 中，保障金鑰安全性。

---

### 📦 部署說明

#### 步驟一：環境設定
1.  在 [Google Apps Script](https://script.google.com/) 建立新專案。
2.  建立 `Code.gs` 並貼入後端處理邏輯。
3.  建立 `index.html` 並貼入前端使用者介面代碼。

#### 步驟二：API 金鑰配置
1.  點擊專案設定（齒輪圖示）。
2.  在「指令碼屬性」中新增：
    *   **屬性名稱**：`GEMINI_API_KEY`
    *   **屬性值**：填入來自 Google AI Studio 的 API Key。

#### 步驟三：發佈
1.  點擊「部署」>「新部署」。
2.  類型選擇「網頁應用程式」。
3.  「執行身分」設為「我」，「誰可以存取」設定為「所有人」。

### 📖 教學應用建議
1.  **語文修辭練習**：結合課文中的「擬人法」與「譬喻法」，讓學生練習將修辭轉化為色彩。
2.  **情緒詞彙擴充**：透過標籤選擇，讓學生認識更多描述感受的詞彙（如：生機、遼闊、平穩）。
3.  **數位與實體整合**：學生完成數位調色後，可將 AI 點評的金句抄錄至紙本學習單中，達成手寫練習的效果。
